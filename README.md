# smart-kitchen-alert


### DHT11 to ESP8266 (NodeMCU) Wiring

| DHT11 Pin | ESP8266 (NodeMCU) Pin |
|-----------|------------------------|
| VCC       | 3.3V (or 5V)          |
| GND       | GND                   |
| DATA      | D4 (GPIO2)            |

### Smoke Sensor (MQ-2 / MQ-135) to ESP8266 (NodeMCU) Wiring

| Smoke Sensor Pin | ESP8266 (NodeMCU) Pin |
|------------------|------------------------|
| VCC              | 3.3V                  |
| GND              | GND                   |
| DO (Digital Out) | D5 (GPIO14)           |
| AO (Analog Out)  | A0 (ADC pin)          |
